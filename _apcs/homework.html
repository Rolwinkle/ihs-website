Lab projects

  1. Evaluate following expressions (this should be done in your notebook)
    • 5 + 2 + "(1 + 1)" + 4 + 2 * 3
    • 89 % 10 / 4 * 2.0 / 5 + (1.5 + 1.0 / 2) * 2
  2. Write a Java program that prints a bill including sales tax. Your program must:
    • Declare and initialize a variable for the subtotal
    • Declare and initialize a variable for the sales tax rate
    • Use these variables throughout the rest of the program
    • Print the results in the format illustrated below
      • (Your program need not worry about properly formatting monetary totals or round-off errors.)
    Example output: Subtotal: $26.34 Tax (9.5%): $2.5023 Total: $28.8423
  3. Rewrite your "three boxes" program.
    1. Remove some copied code and make it easier to change.
    2. Use a variable to control the "height" of the box.
    3. Use a variable to control the number of boxes.
